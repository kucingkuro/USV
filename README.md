**Unmanned Surface Vehicle - Ranger**

Open source unmanned surface vehicle project.

*Tech Stack*

*Hardware*
- ELRS ER5A Receiver
- Brushless Motor A2212 KV2000
- ESC Simonk 20A
- Servo 9SG
- LiPo Battery 2200MAh
- Action Cam B-PRO 5

*Structural*
- 3D Printed PLA
